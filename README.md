# 🚗 Automobile Sales Statistics Dashboard

An interactive data visualization dashboard built with Python and Plotly Dash to analyze historical automobile sales trends during recession and non-recession periods.

## 🔗 Live Demo
👉 [View the Live Dashboard](https://automobile-dashboard-y8b0.onrender.com)

## 📊 Overview
This dashboard allows users to explore automobile sales data from 1980 to 2023, comparing trends during recession periods vs normal economic conditions. Users can interactively switch between two report types and filter by year.

## ✨ Features
- **Recession Period Statistics** — Analyze how recessions impacted automobile sales
- **Yearly Statistics** — Explore sales trends for any specific year from 1980 to 2023
- Interactive dropdowns to filter data dynamically
- 8 interactive charts including line charts, bar charts, and pie charts

## 📈 Charts Included
### Recession Report
- Automobile sales fluctuation over recession years (Line Chart)
- Average vehicles sold by vehicle type during recession (Bar Chart)
- Advertising expenditure share by vehicle type (Pie Chart)
- Effect of unemployment rate on vehicle sales (Bar Chart)

### Yearly Report
- Yearly automobile sales for the whole period (Line Chart)
- Total monthly automobile sales (Line Chart)
- Average vehicles sold by vehicle type in selected year (Bar Chart)
- Total advertising expenditure by vehicle type (Pie Chart)

## 🛠️ Technologies Used
- **Python** — Core programming language
- **Plotly Dash** — Interactive web dashboard framework
- **Plotly Express** — Data visualization library
- **Pandas** — Data manipulation and analysis

## 📂 Dataset
The dataset contains historical automobile sales data including:
- Automobile Sales
- Vehicle Type
- Recession indicators
- GDP, Unemployment Rate
- Consumer Confidence
- Advertising Expenditure
- Seasonality Weight

## 🚀 How to Run Locally
1. Clone the repository
```bash
git clone https://github.com/myleschukwukaz-create/automobile-dashboard.git
cd automobile-dashboard
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the app
```bash
python "dash project/Dash project.py"
```

4. Open your browser and go to `http://localhost:8050`
